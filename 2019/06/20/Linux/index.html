  - vim 中翻页：向上查找和向下查找字符串的命令是
    - j --向下
    - k-- 向上
    - h--向左
    - l--向右
    - ctrlb--向上移动一屏
    - ctrlf--向下移动一屏
    - 上下左右箭头也可以
  - 启动linuxshell的时候，stdin.stdout,stderr,三个文件描述符会自动打开，他们的值分别是：标准输入，标准输出、标准错误
  - 为脚本程序添加执行权限的命令及参数：chmod a+x filename
  - 用八进制表示权限：
