具有iterable类型的集合可以通过新的for…of循环遍历。

for … of循环和for … in循环有何区别
for … in循环，它遍历的实际上是对象的属性名称。

一个Array数组实际上也是一个对象，它的每个元素的索引被视为一个属性。
