React组件间通信

发表于 2017-12-10 | 更新于 2018-08-18 | 分类于 技巧 | 评论数：

处理 React 组件之间的交流方式，主要取决于组件之间的关系，
然而这些关系的约定人就是你。
React 组件之间交流的方式，可以分为以下 3 种：

  • 【父组件】向【子组件】传值；
  • 【子组件】向【父组件】传值；
  • 没有任何嵌套关系的组件之间传值（PS：比如：兄弟组件之间传值）
