git小记
最近遇到一个神奇的问题～
平时推送到远程分支上用 git push
但发现了另一种方法，git push origin master
他们有什么区别呢？

git push origin master 的意思就是，上传本地当前分支代码到master 分支，git push
是上传本地所有分支代码到远程对应分支上。
